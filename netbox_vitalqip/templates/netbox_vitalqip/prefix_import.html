Search VitalQIP Networks

Search VitalQIP for networks to import as prefixes, or look up a single IP address.

Wildcard patterns search for networks/prefixes: 10.*, 172.16.*, 192.168.1.*
Full IP addresses look up a single address with DNS name: 192.168.1.1
How It Works
  1. Enter an address pattern (e.g., 10.* or 192.168.1.*)
  2. Plugin queries VitalQIP for matching networks
  3. Review networks and select which to import
  4. Imported prefixes are created in NetBox with:
    • Description from QIP network name
    • OHSU tenant
    • Status: Container (for large) or Active (for small)
Notes
  • Broader patterns (e.g., 10.*) return more results
  • Networks already in NetBox will be marked as "Exists"
  • VitalQIP responses are cached for 5 minutes
  • Networks marked "DISCONNECTED" are shown but can be skipped
